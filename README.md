# jossycars - Real Cars ETH

Premium luxury automotive landing page and digital menu featuring a 3D model showcase, advanced vehicle filtering, video reviews, client authentication, and administrative inventory management.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS, JavaScript, GSAP, model-viewer (Google 3D)
- **Backend**: Node.js, Express, MongoDB Atlas, JWT Authentication
- **Localization**: Amharic / English toggle with local storage persistence
